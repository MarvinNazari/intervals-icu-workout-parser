## Easy / Recovery

### Easy Run

Conversational pace, build aerobic base.

```
Easy Run

- 45m Z2 HR
```

### Recovery Run

Very easy, post-hard-workout recovery.

```
Recovery Run

- 30m Z1 HR
```

## Tempo

### Classic Tempo

Sustained comfortably hard effort.

```
Tempo Run

Warmup
- 10m Z2 HR

Main Set
- 20m 85-90% pace

Cooldown
- 10m Z2 HR
```

### Tempo Intervals

Broken tempo for beginners.

```
Tempo Intervals

Warmup
- 10m Z2 HR

Main Set 3x
- 8m 88% pace
- 4m Z2 HR

Cooldown
- 10m Z1 HR
```

## Threshold

### Cruise Intervals

Classic Daniels-style threshold work.

```
Cruise Intervals

Warmup
- 15m Z2 HR

Main Set 4x
- 5m 95-100% pace
- 90s Z1 HR

Cooldown
- 10m Z2 HR
```

### Threshold Run

Sustained threshold effort.

```
Threshold Run

Warmup
- 15m 70% pace

Main Set
- 20m 95-100% pace

Cooldown
- 10m 65% pace
```

## VO2 Max / Intervals

### 800m Repeats

Classic VO2 max intervals.

```
800m Repeats

Warmup
- 15m Z2 HR
- 4x 100m 90% pace
- 4x 100m Z1 HR

Main Set 5x
- 800m 105-110% pace
- 400m Z1 HR

Cooldown
- 10m Z1 HR
```

### Hill Repeats

Build strength and VO2 max.

```
Hill Repeats

Warmup
- 10m 70-75% pace

Main Set 8x
- 45s 95-100% pace
- 2m 60-70% pace

Cooldown
- 10m 70% pace
```

### 400m Repeats

Short, fast intervals.

```
400m Repeats

Warmup
- 15m Z2 HR

Main Set 8x
- 400m 110% pace
- 400m Z1 HR

Cooldown
- 10m Z1 HR
```

## Long Run

### Aerobic Long Run

Build endurance.

```
Long Run

- 90m Z2 HR
```

### Progressive Long Run

Finish faster than you started.

```
Progressive Long Run

- 30m 70% pace
- 30m 75% pace
- 20m 80% pace
- 10m 85% pace
```

### Long Run with Tempo

Build race-specific endurance.

```
Long Run with Tempo

- 30m Z2 HR
- 20m 85-90% pace
- 20m Z2 HR
- 10m 88% pace
- 10m Z2 HR
```

## Speed Work

### Strides

Neuromuscular activation.

```
Easy Run with Strides

Main
- 30m Z2 HR

Strides 6x
- 20s 110% pace
- 40s Z1 HR
```

### Fartlek

Unstructured speed play.

```
Fartlek

Warmup
- 10m Z2 HR

Main Set
- 3m 90% pace
- 3m Z2 HR
- 2m 95% pace
- 2m Z2 HR
- 1m 100% pace
- 1m Z2 HR
- 2m 95% pace
- 2m Z2 HR
- 3m 90% pace
- 3m Z2 HR

Cooldown
- 10m Z1 HR
```
