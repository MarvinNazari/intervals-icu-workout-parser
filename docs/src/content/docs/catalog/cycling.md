## Sweet Spot

### Classic Sweet Spot

The gold standard for building FTP.

```
Sweet Spot

Warmup
- 10m 50-65% 90rpm
- 5m ramp 65-85%
- 2m 55% 85rpm

Main Set 3x
- 10m 88-94% 90rpm
- 5m 55% 85rpm

Cooldown
- 10m ramp 55-40% 85rpm
```

### Sweet Spot Over-Unders

Builds fatigue resistance.

```
Sweet Spot Over-Unders

Warmup
- 10m 60%
- 5m ramp 60-85%

Main Set 3x
- 2m 95%
- 4m 88%
- 2m 95%
- 4m 88%
- 4m 55%

Cooldown
- 10m 50%
```
