## Technique / Drill

### Technique Focus

Drill-heavy session for form work.

```
Technique Session

Warmup
- 300m Z2 pace
- 200m drill

Main Set 4x
- 50m drill
- 100m Z2 pace

Cooldown
- 200m Z1 pace
```

### Catch-Up Drill Set

Focus on catch timing.

```
Catch-Up Drills

Warmup
- 400m Z2 pace

Drill Set 4x
- 50m catch-up drill
- 50m fingertip drag
- 100m Z2 pace

Cooldown
- 200m Z1 pace
```

## Endurance

### Base Endurance

Steady aerobic swimming.

```
Base Endurance

Warmup
- 300m easy
- 200m pull

Main Set 4x
- 200m 75% pace 20s rest
- 100m 70% pace 15s rest

Cooldown
- 200m easy
```

### Long Swim

Build aerobic capacity.

```
Long Swim

Warmup
- 400m Z2 pace

Main Set
- 1500m 75% pace

Cooldown
- 200m Z1 pace
```

## Threshold

### CSS Intervals

Critical Swim Speed development.

```
CSS Training

Warmup
- 400m Z2 pace
- 4x 50m 85% pace

Main Set 5x
- 200m 95% pace 30s rest

Cooldown
- 200m Z1 pace
```

### Threshold 100s

Short threshold intervals.

```
Threshold 100s

Warmup
- 300m Z2 pace
- 200m drill

Main Set 8x
- 100m 95-100% pace 20s rest

Cooldown
- 200m Z1 pace
```

## Speed

### Sprint Set

Develop speed and power.

```
Sprint Training

Warmup
- 400m Z2 pace
- 4x 25m build

Main Set 8x
- 25m 110% pace 30s rest
- 75m Z2 pace

Cooldown
- 200m Z1 pace
```

### Descend Set

Negative split each repeat.

```
Descend Set

Warmup
- 300m Z2 pace

Main Set 3x
- 100m 80% pace
- 100m 85% pace
- 100m 90% pace
- 100m 95% pace
- 100m Z1 pace

Cooldown
- 200m Z1 pace
```

## Mixed / Race Prep

### Race Simulation

Triathlon swim prep.

```
Race Simulation

Warmup
- 300m easy
- 4x 50m build

Main Set
- 100m 100% pace
- 400m 90% pace
- 100m 95% pace
- 200m Z2 pace

Cooldown
- 200m easy
```

### IM Prep

Individual Medley practice.

```
IM Practice

Warmup
- 400m free Z2 pace

Main Set 4x
- 50m fly 85% pace
- 50m back 85% pace
- 50m breast 85% pace
- 50m free 85% pace
- 100m Z1 pace

Cooldown
- 200m easy
```

Recovery Swim
