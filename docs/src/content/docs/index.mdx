## Quick Example

```typescript
import { parseWorkoutStructure, WorkoutBuilder } from 'intervals-icu-workout-parser'

// Parse existing workout
const structure = parseWorkoutStructure(`
Tempo Run

Warm Up
- 10m Z2 HR

Main Set 3x
- 10m 85-90% pace
- 5m Z2 HR

Cool Down
- 10m Z1 HR
`)

// Build a new workout
const workout = new WorkoutBuilder()
  .title('Sweet Spot')
  .section('Main Set').repeat(3)
    .step().time(8).powerRange(88, 94).add()
    .step().time(4).power(55).add()
  .buildWorkout()
```
